這個站是用來**整理技術內容**的：以乾淨的靜態頁面為主，必要時在文章裡嵌入互動動畫、互動元件與結構化表格，把那些「文字很難講、一動起來就懂」的概念說清楚。每篇文章底部都能留言。整站部署在 GitHub Pages，零後端、零維運成本。

## 用什麼搭的

- **Astro**：以靜態輸出為主，預設零 JavaScript。
- **Astro Islands**：只有需要互動的元件才會「注水」載入 JS，其餘都是純 HTML/CSS。
- **MDX 內容**：文章是 Markdown，但可以直接 `import` 並嵌入元件。
- **GitHub Pages + Actions**：push 到 `main` 就自動建置部署。

## 怎麼新增一篇文章

1. 在 `src/content/posts/` 新增一個 `.mdx`（或 `.md`）檔，檔名就是網址 slug。
2. 最上面寫 frontmatter，欄位如下：

| 欄位          | 必填 | 說明                        |
| ------------- | ---- | --------------------------- |
| `title`       | ✓    | 文章標題                    |
| `description` | ✓    | 列表與 SEO 用的一句話簡述   |
| `pubDate`     | ✓    | 發佈日期（`YYYY-MM-DD`）    |
| `updatedDate` |      | 更新日期                    |
| `tags`        |      | 標籤陣列，例如 `[Linux, C]` |
| `draft`       |      | `true` 時不會出現在正式站   |
| `toc`         |      | 是否顯示目錄，預設 `true`   |

3. 正文用 Markdown 寫；要嵌入互動元件就在 frontmatter 下方 `import` 進來再使用。存檔，`pnpm dev` 會即時重新整理。

> 小提醒：站內所有連結請走 `withBase()` helper（見 `src/utils/path.ts`），這樣切換到子路徑部署（`/my-pages/`）也不會破圖。

## 在文章裡嵌入互動動畫

文章可以嵌入兩類動畫。第一類是**重互動 / 3D**，用 Three.js 包成 island，以 `client:visible` 載入——捲到這裡才會初始化，而且尊重「減少動態效果」偏好：

<ThreeDemo client:visible />

用滑鼠拖曳可以轉動它。離開頁面時元件會正確 `dispose`，不留記憶體。

第二類是**輕量、零 JS** 的純 CSS/SVG 動畫，適合訊號、資料流之類的示意：

<SignalWave />

至於這個站最主要的用途——**逐 byte 的演算法 / 記憶體動畫**——可以看〈[strncpy() 在 Linux kernel 的退場](/posts/strncpy-and-the-kernel/)〉，整篇都是互動動畫。

## 表格：靜態與可排序兩種

直接寫 Markdown 表格就有不錯的樣式（斑馬紋、手機可橫向捲動、表頭 sticky）。如果需要**可點欄位排序**，可以用 `<DataTable>` 元件，傳 JSON 進去即可。下面比較幾種留言系統（點欄位標題排序看看）：

<DataTable
  client:visible
  columns={[
    { key: 'name', label: '系統' },
    { key: 'storage', label: '資料存放' },
    { key: 'selfhost', label: '可自架' },
    { key: 'weight', label: '前端負擔' },

  rows={[

    { name: 'Remark42', storage: '自架資料庫', selfhost: '是', weight: 2 },
    { name: 'Artalk', storage: '自架資料庫', selfhost: '是', weight: 2 },

/>

這個站用 **Giscus**（以 GitHub Discussions 為後端）。它的好處是零維運、登入即用；代價是綁在 GitHub 上。為了不被綁死，倉庫裡附了一支 `scripts/export-comments.mjs`，可以隨時把所有留言匯出成 JSON 與 Disqus 相容格式，將來要轉自架（Cusdis / Remark42 / Artalk）時用得到。
